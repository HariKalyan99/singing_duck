# Prompt2App - AI App Builder SaaS Template

Prompt2App is a modern AI website builder SaaS template that lets users generate full web apps from natural language prompts. It includes promt box in hero section, login page, animated landing sections, pricing, and responsive layouts designed for startups and AI builders.

**Live Preview:**
https://prompt2app.prebuiltui.com

## Installation Guide


### React

1. Navigate to the `/reactjs` directory.
2. Install dependencies:

   npm install

3. Run development server:

   npm run dev
